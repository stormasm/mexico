### How to get there via Bus

- Metro #1 Line to the San Lazaro Stop
- Take the bus from TAPO
- ADO bus line has departures that leave every 20 minutes
- The ride is 2 hours
- Uber can not pick up at station so walk a few blocks
- Uber to Zocalo is 160 pesos
